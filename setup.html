Next Steps

  1. Create a new GitHub repository (or use an existing one).
  2. Enable GitHub Pages: repo Settings → Pages → Source: Deploy from branch → main → / (root)
  3. Upload all the files from this project to the repo root (including responses/.gitkeep).
  4. Use this setup page to save your config.txt to GitHub, or download and commit it manually.
  5. Share your GitHub Pages URL (https://YOUR_USERNAME.github.io/YOUR_REPO/) with friends!
